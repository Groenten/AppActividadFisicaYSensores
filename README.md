# App Actividad Física y Sensores

Aplicación Android desarrollada en Kotlin como práctica para el uso de sensores del dispositivo.

## Funcionalidades
- Registro de actividades físicas (caminar, correr, yoga, gimnasio, etc.)
- Selección del tipo de actividad mediante Spinner
- Cuenta atrás de la sesión mediante temporizador
- Detección de movimiento con acelerómetro (semáforo de intensidad)
- Medición de distancia recorrida mediante GPS
- Reproducción de sonido al finalizar la sesión

## Tecnologías utilizadas
- Kotlin
- Android Studio
- Sensores: Acelerómetro
- GPS / LocationManager
- RecyclerView (en versiones previas)
- CountDownTimer

## Uso
1. Seleccionar el tipo de actividad
2. Introducir la duración en minutos
3. Iniciar la sesión
4. Mover el dispositivo para ver la intensidad del movimiento
5. Finalizar la sesión para ver el resumen
